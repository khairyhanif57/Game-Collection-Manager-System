# Game-Collection-Manager-System

Designed and developed a Python-based GUI application integrated with a MySQL database for managing video game collections. The system allows users to add, update, delete, and view game records through an interactive interface built using Tkinter. The project demonstrates the implementation of CRUD operations, database connectivity, and user-friendly software design concepts.

Programming Language Used
- Python
- MySQL
- Tkinter

How to Run/Open
1. Install Python and MySQL.
2. Create the required database and tables in MySQL.
3. Install the mysql-connector-python package.
4. Open the project in VS Code or any Python IDE.
5. Run the Python file to launch the application.
